# EduFace Attendance

A production-ready Face Recognition Attendance System for colleges using Gemini AI for high-accuracy identity verification.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
